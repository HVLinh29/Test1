Having luxurious, eye-catching design with rounded edges and white/black color, Lumi’s infrared controller is not only the key device in air conditioner and smart TV solutio but also an important feature to make your home more impressive. Moreover, the device is made from PP non flammable plastic which is highly appreciated by international engineers and experts APPLICATION

Infrared control system has the ability to replace traditional remote controls of air conditioners, TV fans, digital players, … and receive and learn from over 1,000 commands. The device contains libraries of more than 300 codes available for 4 control categories: TV, Fan, air conditioner and digital player.
Thanks to the combination with the central control unit, you can schedule on / off smart air conditioning to turn on the air conditioner at 10 pm and turn off at 4 am, etc. Additionally, the intelligent infrared controller also has integrated sensors for measuring environmental parameters to increase or decrease the temperature based on the external environment.

With an infrared control device, it is possible to create multiple interfaces on the phone to control multiple devices at the same time. For example, appliances in the living room can be set up to create a console from what is available in the library. In case the library does not support it, it is also possible to set the “learning” mode for the device.

Note: Place the infrared control device within 5m away from the device controlled.
With the cooperation between Google Assistant voice control and Lumi solution, now you just need to say “OK Google, turn on the air conditioner” or “OK Google, turn on the TV”, immediately the TV and air conditioner will be activated as you want.

Moreover, the 2019 infrared controller is much more improved with the library function, which enables you to control the device via your phone similar to a remote control.

OPERATION

The infrared controller will take control commands from the library or learn all the commands from remote controls of all devices in your house. On the user’s phone app will appear icons such as: air conditioner, TV, fan so instead of using Remote, users will directly press icons on their phone to control the device.
Each device will have a different interface.

Outstanding advantages of the infrared controller

Instead of 10 remote controls, you only need one phone to completely control the devices in the house.
On the other hand, when users install the context of “going home” – a system of devices including air conditioner, water heater, lights, curtains and so on will be ready to serve you as soon as you arrive home.

The way to install Lumi infrared controller is very simple, you can install it yourself at home:

Installation

Step 1: Power on the device

  • Use the included power adapter to power on the device
  • The device has two power locations: the bottom or side of the device
  • After being powered on, the indicator light on the device will flash red three times and then turn red

 

Step 2: Install the device
The LM-IRZ device is designed with two options:

  • Place on a table or flat surface near the infrared receiver and the devices you want to control.
  • Mount the LM-IRZ 2.0 device on a wall by tightening the expansion screws that came with it and then mounting it such that the screw hole on the back of the device fits into the head of the expansion screw.

Note: The distance from the remote control to the devices must be less than 5 meters.

Setting up device on the Lumi Life+ app

Joining the network

  • To set up the IR remote control using the app on your phone, you need to have a HC central controller and have created a home on the system.
  • Log in to the Lumi Life+ app => select Settings => select Device settings => select HC => Zigbee => allow the device to join the network.
  • Observe the indicator light on the device. If the device joins the network successfully, the indicator light will flash red three times in a cycle of once per second.
  • If the device does not join the network, the switch may have been configured. In this case, you need to reset the device to return to the network joining mode.
  • When the device joins the network successfully, the IR 2.0 remote will display two types of controls as shown below:
  • Type 1: Learning commands from a handheld remote control to control the device.
  • Type 2: Controlling using library commands available on the Lumi server.

Setting up the type of control using the library

Step 1:

  • The user selects the available library control => Select “Add remote” to add the control you want to use.

Step 2:

  • The user selects the type of control they want to use.
  • For available command controls, Lumi supports 4 types: TV, Fan, AC, and Set top box.

Step 3:

  • Select the brand of the device you selected in step 2.

Step 4:

  • Try to control the device using the commands displayed on the screen to check if the device can be controlled.

Test controlling the device

  • If the device can be controlled, the user selects “Yes”, proceeds to name the control and puts the control into room installation the IR.
  • If the signal is not received, press “No” to continue trying a different type.

Setting up the type of control that learns commands from a handheld remote

Step 1: Create a Learning Remote (Type 1)

  • The user selects the Learning Remote type.
  • For the Learning Remote type, it supports 3 types of devices: TV, Air Conditioner, and Fan.

Step 2: Put the device into learning mode

  • The newly created remote will be displayed in the IR tab.
  • Tap the icon of the newly created remote, then proceed to learn the command.

Step 3: Learn commands from the remote’s buttons

  • By default, all icons are hidden or do not have control command information. The user presses the command icons on the remote to proceed with learning the command.
  • Select “Learn Command” on the screen to display the learning command status.
  • The user points the device’s remote control towards the IR remote and presses the command button to be learned.
  • During the learning process, the user can press “Cancel” to cancel the IR learning status.
  • After learning the command, the icon that has just been learned will light up, allowing customers to tap to control the device.

Step 4: Name the remote and put the remote into the room

  • Activating infrared commands from the Lumi Life app
  • To control the device, the user taps directly on the corresponding command icon on the app. At this time, the indicator light on the IR remote will flash green twice to indicate that the control command has been sent successfully.

Note:

  • For the type of remote that needs to learn commands, the button on the App must be learned first before the device can be activated.
  • When controlling the device, the remote icon will not change the status on the app.

Deleting learned commands

  • Only applies to the learning remote type.
  • The user holds down the command icon of the command to be deleted => selects “Delete” to proceed with deleting the command or selects “Cancel” to cancel the selection, then returns to the control screen.

In addition, the user can set up the device with advanced features such as:

  • Setting rules
  • Setting scenes
  • Setting schedules

Reset the device

The user holds down the Config button for 5 seconds (round plastic button on the bottom) until the indicator light turns red, then releases the button. The device will leave the network and reset. The device will reset successfully and return to its original state.

Packaged product

The product packaging box includes: 1 IR remote control, 1 user manual, 1 5V/1A adapter, and 1 set of mounting screws.
